# Portfolio développeur

Portfolio professionnel développé avec React, Vite et Sass.

## Présentation

Ce projet correspond à la création de mon portfolio de développeur dans le cadre de ma formation OpenClassrooms.

Le site présente :

- mon profil de développeur front-end React freelance ;
- mes compétences techniques ;
- une sélection de projets réalisés pendant ma formation ;
- mes liens de contact ;
- un site responsive, accessible et optimisé.

## Site en ligne

[Voir le portfolio](https://portfolio-d-veloppeur.vercel.app/)

## Objectifs du projet

- concevoir un portfolio professionnel ;
- mettre en valeur plusieurs projets réalisés pendant la formation ;
- créer une interface moderne, responsive et maintenable ;
- appliquer de bonnes pratiques de SEO et d’accessibilité ;
- publier le site en ligne.

## Technologies utilisées

- React
- Vite
- Sass
- React Icons

## Projets présentés

- **Kasa**  
  Application de location immobilière développée avec React, React Router et Sass.

- **Sophie Bluel**  
  Site dynamique en JavaScript avec communication avec une API.

- **Débuggez le site d'une agence d'événementiel**  
  Correction de bugs, vérification du bon fonctionnement et amélioration de la qualité du site.

- **Application bancaire**  
  Front-end bancaire développé avec React, Redux et API REST.
